# 使い方

このページでは、sumstockの基本的な使い方について説明します。

## 基本的な操作

### データの取得

スムストックからデータを取得する方法：

```python
# サンプルコード（例）
from sumstock import DataFetcher

fetcher = DataFetcher()
data = fetcher.get_data()
print(data)
```
