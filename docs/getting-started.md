# はじめに

sumstockへようこそ！このガイドでは、プロジェクトの基本的な使い方を説明します。

## 前提条件

このプロジェクトを使用するには、以下が必要です：

- Python 3.x またはその他の必要な環境
- 基本的なコマンドラインの知識

## インストール

```bash
# リポジトリをクローン
git clone https://github.com/zGVfRcgR/sumstock.git

# ディレクトリに移動
cd sumstock
```
